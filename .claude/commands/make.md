# プロジェクトビルドパイプライン

このコマンドは以下の処理を順番に実行します：

1. **フォーマット** - コードフォーマットを適用
2. **リント** - コード品質チェック
3. **タイプチェック** - TypeScript型チェック
4. **テスト** - 全テストスイートの実行
5. **ビルド** - プロダクションビルド

## 実行

```bash
# 1. フォーマット
pnpm run format

# 2. リント
pnpm run lint

# 3. タイプチェック
pnpm run typecheck

# 4. テスト
pnpm run test

# 5. ビルド
pnpm run build
```

各ステップでエラーが発生した場合は処理を停止し、エラー内容を報告します。
